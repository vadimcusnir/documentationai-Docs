## Latest Updates

  CUSNIR OS releases follow semantic versioning. You receive automatic notifications for new versions in the dashboard. Check your current version in the settings panel.
</Callout>

## How to Update

Follow these steps to update CUSNIR OS to the latest version.

<Steps>
  <Step title="Check Current Version" icon="info">
    Navigate to the dashboard settings and view your installed version.
  </Step>
  <Step title="Download Update" icon="download">
    <CodeGroup tabs="Docker,CLI">
      ````bash
      docker pull cusnir/cusnir-os:latest
      ````

      ````bash
      cusnir-cli update --latest
      ````
    </CodeGroup>
  </Step>
  <Step title="Restart Service" icon="refresh-cw">
    Restart the CUSNIR OS container or service to apply changes.

    ````bash
    docker restart cusnir-os-container
    ````
  </Step>
  <Step title="Verify Update" icon="check-circle">
    Confirm the new version in the settings panel.
  </Step>
</Steps>
