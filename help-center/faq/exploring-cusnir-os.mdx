## Overview

Yes, you can access and explore the CUSNIR OS interface without creating an account. Demo mode lets you view core features freely.

## Demo Limitations

- Some advanced tools require login.
- Changes made in demo mode are not saved.
- Governance actions are unavailable.

## How To Try Demo

Simply go to `https://cusniros.com/demo` and interact with the platform.

<Callout kind="info">
To fully experience decision-making features, create a free account.
</Callout>
